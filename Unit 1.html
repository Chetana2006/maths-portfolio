Unit 1: Description of Data

Data refers to a collection of facts, figures, or observations collected for analysis. The description of data involves organizing, summarizing, and presenting data in a meaningful way.

Types of Data

  • Qualitative Data – non-numerical data such as gender and color
  • Quantitative Data – numerical data such as height and marks
  • Discrete Data – countable values
  • Continuous Data – measurable values within a range

Presentation of Data

Data can be presented using tables, bar charts, histograms, and pie charts to simplify interpretation.

Importance

Describing data helps in identifying patterns, trends, and serves as the foundation for statistical analysis.
